# Glossary

| Term / Slug          | Meaning                                                               |
|----------------------|-----------------------------------------------------------------------|
| ABC Nepal TV         | Brand of the news organization.                                       |
| ABCountability / abcnepal | Function prefix used across this theme.                          |
| abc_news             | Theme / project name.                                                 |
| abc-video            | Category slug for video section.                                      |
| artha                | Nepali label for business / economics.                                |
| cpt                  | Custom post type.                                                     |
| live_update          | Custom post type used for live-blog entries.                          |
| kutniti              | Nepali label for diplomacy section.                                   |
| mn_section           | Helper closure in `index.php` for a section card row.                 |
| mn_*                 | CSS prefix used on the Main News page.                                |
| nav-toggle           | Legacy checkbox-class toggle (hidden).                                |
| ndc-today-date       | Shortcode from Nepali Date Converter plugin.                          |
| pradesh              | Province section.                                                     |
| section config       | Static associative array returned by `abcnepal_section_config()`.      |
| sp_*                 | CSS prefix used on the Single Post template.                          |
| bnn_*                | CSS prefix used by `breaking-banner.php`.                             |
| template part        | Partial PHP file included by a template (Theme does not currently use `get_template_part`; it uses `include()`). |
| update_topic         | Hierarchical taxonomy for `live_update`.                              |
| WP_Query             | Core WordPress database query class.                                  |
| wp_reset_postdata()  | Cleanup function after custom WP_Query loops.                         |
